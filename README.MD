# Реестр Web Упрощение

Этот проект представляет собой веб-инструмент для упрощения работы с реестром данных. Он создан, чтобы заменить ручное заполнение Excel таблиц и ускорить процесс ведения записей.

## О проекте

Мы берем данные из Excel и переносим их в веб. Сайт автоматизирует рутину: вам больше не нужно постоянно прокручивать таблицу в поисках строки или бояться случайно испортить формулу.

**Как это работает:**
1.  Данные загружаются из исходного Excel-файла.
2.  Веб-интерфейс отображает реестр в удобном и понятном виде.
3.  Через сайт можно добавлять новые записи, редактировать существующие или удалять их.
4.  Все изменения автоматически сохраняются (синхронизируются с данными), что позволяет вести актуальный реестр без лишних телодвижений.

## Цель

Сделать процесс ведения реестра быстрым, понятным и доступным для всех сотрудников, минимизировать ошибки человеческого фактора и избавиться от необходимости работы напрямую с громоздкими Excel-файлами.
